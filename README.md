# Sistema Concessionária

Projeto desenvolvido em Java com foco no gerenciamento de uma concessionária de veículos, permitindo o controle de automóveis, clientes, vendas e operações administrativas.

## 📋 Sobre o Projeto

O **Sistema Concessionária** foi criado com o objetivo de praticar conceitos de:

* Programação Orientada a Objetos (POO)
* Estruturas de dados
* Organização em camadas
* Manipulação de informações em Java
* Boas práticas de desenvolvimento

O sistema simula o funcionamento básico de uma concessionária, permitindo realizar operações de cadastro e gerenciamento de veículos e clientes.

## ⚙️ Funcionalidades

* Cadastro de veículos
* Cadastro de clientes
* Listagem de veículos
* Controle de vendas
* Busca de informações
* Organização orientada a objetos


## 🧠 Conceitos Aplicados

Este projeto utiliza diversos conceitos importantes do desenvolvimento Java:

* Classes e Objetos
* Encapsulamento
* Herança
* Polimorfismo
* Collections
* Organização em pacotes
* Boas práticas de código

## 📚 Objetivo do Projeto

O principal objetivo deste projeto é servir como prática de estudos e evolução no desenvolvimento backend com Java.
